Display Star Wars character data

To get started using the SWAPI API, we'll create a simple page that displays Star Wars character data in an interactive table. For this example, we'll collect the character's Name, Birth year (Star Wars year), Homeworld and the Films they appeared in.

We'll also give the table simple styling and a button to get even more data in those eyeballs and quite possibly find the droids you were looking for.

What you'll need​

To complete this first part of the tutorial, you'll need the following:

  • A basic understanding of HTML, JavaScript, and REST APIs
  • An empty repository or somewhere to save your site (we're using GitHub)
  • A tool like Notepad++ or Visual Studio Code to edit files locally
  • A continued interest in Star Wars

💡 This tutorial assumes you're starting from scratch. However, the code can be copied and inserted into an existing site or web app. Just skip the set up.

Set up the site structure​

In the root of your repository,

  1. Create a new file named index.html
  2. Create a new file named table.css
  3. Create a new folder named scripts
  4. Open the scripts folder
  5. Create a new file named main.js
  6. Save the files and folders

You should see the following:

#Demo site structure
./root/*
├─ scripts
|   ├─ main.js

Add a HTML table​

Copy this HTML into your index.html file. It includes the basic syntax for using the SWAPI data in a four-column table.

The rows will be populated using the data we fetch and pass into the id=data value.


<!DOCTYPE html>
<html>
    <head>
        <title>Star Wars database</title>
        <meta charset="UTF-8" />
        <script src="scripts/main.js"></script>
        <link href="table.css" rel="stylesheet">
    </head>
<body onload="fetchCharacters()">
    <h1>Star Wars Characters</h1>
        <table id="characters">
            <thead>
                <tr>
                    <th>Name</th>
                    <th>Birth year</th>
                    <th>Homeworld</th>
                    <th>Films</th>
                </tr>
            </thead>
            <tbody id="data"></tbody>
        </table>
    <button onclick="fetchPage(2)">Load Page 2</button>
</body>
</html>

Setup the API call​

Using this script, we will call the people resource and fetch the character's name, birth_year, homeworld, and films. You can write this from scratch using a different language or copy the example JavaScript.

As SWAPI stores the homeworld attribute as a URL, you'll need to define a second function to fetch it's name attribute and append it to the URL.

Similarly, the films attribute is stored as an array of URLs, so you'll need two additional functions to loop through these and join and append them.

💡 You should copy all the code in these steps into your "main.js" file that we created earlier. You can find the complete script here.

  1. Define a function to fetch the character data. We have called ours fetchCharacters

        // Fetch and display Star Wars character data.
        async function fetchCharacters()
            {
                // Fetch people data and set "temp" variable
                let response = await fetch("https://swapi.dev/api/people/?page=1");
                const data = await response.json();
                let temp = "";

  2. Add a nested function to fetch the homeworld name attribute

                // Fetch the homeworld name
                const getHomeworldName = async (homeworldUrl) => {
                    let homeworldResponse = await fetch(homeworldUrl);
                    let homeworldData = await homeworldResponse.json();
                    return homeworldData.name
                }

  3. Add two more to fetch each film's name and then join them into a comma-separated list

                /* Film Names */
                // Fetch film name
                const getFilmName = async (filmUrl) => {
                    let filmResponse = await fetch(filmUrl);
                    let filmData = await filmResponse.json();
                    return filmData.title;
                };

                //Fetch and display film names
                const getFilmNames = async (filmUrls) => {
                    const filmNames = [];
                    for (const filmUrl of filmUrls) {
                        const filmName = await getFilmName(filmUrl);
                        filmNames.push(filmName);
                    }
                    return filmNames.join(', ');
                };

  4. Loop through the fetched character data and add it to the table

                // loop through each character and set table data
                for (const character of data.results)
                {
                    temp += "<tr>";
                    temp += "<td>" + character.name + "</td>";
                    temp += "<td>" + character.birth_year + "</td>";
                    const homeworldName = await getHomeworldName(character.homeworld);
                    temp += "<td><a href='" + character.homeworld + "' target='_blank'>" + homeworldName + "</a></td>";
                    const filmNames = await getFilmNames(character.films);
                    temp += "<td>" + filmNames + "</td>";
                    temp += "</tr>";
                };
            document.getElementById("data").innerHTML = temp;
            }

  5. Add a function for the next page button

            //Fetch page//
            async function fetchPage(page) {
                await fetchCharacters(page);
            }

Completed script​

Your finished script should look something like this

// Fetch and display Star Wars character data.
async function fetchCharacters()
   {
           // Fetch people data and set "temp" variable
           let response = await fetch("https://swapi.dev/api/people/?page=1");
           const data = await response.json();
           let temp = "";

           // Fetch the homeworld name
           const getHomeworldName = async (homeworldUrl) => {
               let homeworldResponse = await fetch(homeworldUrl);
               let homeworldData = await homeworldResponse.json();
               return homeworldData.name
           }


           /* Film Names */
           // Fetch film name
           const getFilmName = async (filmUrl) => {
               let filmResponse = await fetch(filmUrl);
               let filmData = await filmResponse.json();
               return filmData.title;
           };

           //Fetch and display film names
           const getFilmNames = async (filmUrls) => {
               const filmNames = [];
               for (const filmUrl of filmUrls) {
                   const filmName = await getFilmName(filmUrl);
                   filmNames.push(filmName);
               }
               return filmNames.join(', ');
           };

           // loop through each character and set table data
           for (const character of data.results)
           {
               temp += "<tr>";
               temp += "<td>" + character.name + "</td>";
               temp += "<td>" + character.birth_year + "</td>";
               const homeworldName = await getHomeworldName(character.homeworld);
               temp += "<td><a href='" + character.homeworld + "' target='_blank'>" + homeworldName + "</a></td>";
               const filmNames = await getFilmNames(character.films);
               temp += "<td>" + filmNames + "</td>";
               temp += "</tr>";
           };
       document.getElementById("data").innerHTML = temp;
       }

//Fetch page 2//
async function fetchPage(page) {
           await fetchCharacters(page);
       }

The css​

To add some simple styles to the table

  1. Open table.css in an editor
  2. Enter the following 

    /* table layout */

    table {
        table-layout: fixed;
        width: 100%;
        border-collapse: collapse;
        border: 3px;
        margin-top: 5%;

    }

    th, td {
        border: 1px solid;
        padding: 10px;
        text align: left
    }

    th {
        background-color: #D2B48C;
    }

    td {
        background-color: rgb(207, 174, 141);
    }

    /* table headers */

    thead th:nth-child(1) {
        width: 15%;
        font-weight: bold;
    }

    thead th:nth-child(2) {
        width: 15%;
    }

    thead th:nth-child(3) {
        width: 30%;  
    }

    thead th:nth-child(4) {
        width: 40%;
    }

  3. Save table.css

Viewing the table​
